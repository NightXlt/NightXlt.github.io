Solution

EmmmпјҢжғідәҶеҚҠеӨ©пјҢиҝҷдёҚжҳҜеӣһжәҜйўҳе‘ҖгҖӮLT зҡ„еҲҶзұ»зңҹзҡ„зғӮгҖӮи®Ўз®—еҮәеҮәжүҖжңүдҪҚж•°йғҪдёҚзӣёзӯүзҡ„ж•°еӯ—зҡ„дёӘж•°пјҢеҒҮи®ҫжңүдёҖдёӘ n дҪҚзҡ„ж•°еӯ—пјҢжҲ‘д»¬е·Із»ҸзҹҘйҒ“еүҚ n - 1дҪҚзҡ„дёҚзӣёзӯүзҡ„ж•°еӯ—дёӘж•°пјҢйӮЈд№Ҳn дҪҚuniqueж•°еӯ— dp[n] = dp[n- 1] + (dp[n - 1] - dp[n - 2]) * (10 - (n - 1))

dp[n-1] : еүҚйқў n-1 дҪҚ unique зҡ„ж•°зӣ®

(dp[n - 1] - dp[n - 2]) * (10 - (n - 1)) : з¬¬n дҪҚзӣёеҜ№дәҺ n-1 дҪҚдёҠзҡ„еўһеҠ зҡ„ unique ж•°йҮҸ
